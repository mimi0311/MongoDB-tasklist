## Proyecto Integrador: Bases de datos

* **id**: String
* **titulo**: String
* **contenido**: String
* **estado**: String
   - Valores: "pendiente", "en curso", "completada"
* **fechaCreacion**: Date

## Ejemplos

**Documento 1:**
```json
{
  "id": "5f5b5cd4e7179a4b3fsfda1e",
  "titulo": "Comprar leche",
  "contenido": "Preferiblemente leche deslactosada.",
  "estado": "completada",
  "fechaCreacion": "2023-10-18T14:30:00Z",
}
```
